Projeto Dashboard

Este é um painel administrativo moderno e clean, desenvolvido para visualização de métricas operacionais e gerenciamento de dados.

![Preview do Projeto](./imagens/image.png)

Sobre o Projeto
O projeto consiste em uma interface de Dashboard funcional com navegação lateral (sidebar), cabeçalho de perfil e área de conteúdo principal com cards de resumo e gráficos. 

O foco principal foi o uso de **CSS Grid** para criar um layout robusto e organizado, além de **Variáveis CSS** para facilitar a customização de cores e temas.

Tecnologias Utilizadas
HTML5: Estrutura semântica e ícones SVG.
CSS3:  CSS Grid Layout para a estrutura principal.
 - Variáveis (`:root`) para padronização de cores e espaçamentos.
 - Flexbox para alinhamento interno dos componentes.
Google Fonts: Fonte *Poppins* para uma tipografia moderna.

Funcionalidades
-  Barra Lateral de Navegação: Com ícones e estados de hover.
-  Resumo de Atividades: Cards superiores que mostram estatísticas do dia.
-  Perfil do Usuário: Área de identificação no cabeçalho.
-  Grid de Gráficos: Espaço pronto para visualização de dados e gráficos.

Como visualizar
Para rodar o projeto localmente:
1. Faça o download ou clone este repositório.
2. Certifique-se de que as imagens estejam na pasta `/imagens`.
3. Abra o arquivo `index.html` em seu navegador de preferência.
